# Django template to deploy on heroku

#### Remember to put your secret key in settings.py.

1.	Clone this template
2.	Create a new heroku app
3.	Add env variable in Heroku settings
4.	Connect your github account with heroku and select your cloned repo
5.	Deploy it on heroku.
6.	For running createsuperuser and migrations commands, connect to heroku bash using
```bash
heroku run bash --app [name of your app]
```

For deploying this manually, refer this article [click](https://medium.com/@dev117uday/deploying-django-app-on-heroku-in-few-minutes-e977bf7bf187)
